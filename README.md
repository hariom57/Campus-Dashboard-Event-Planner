# IITR Campus Dashboard

This repository contains **only the frontend** code for the IITR Campus Dashboard.

The frontend is a React application built with Vite, designed to provide a dynamic and mobile-first experience for campus events.

## Live Links

- **Frontend (This App):** [https://campus-dashboard-event-planner.vercel.app/](https://campus-dashboard-event-planner.vercel.app/)
- **Backend API:** [https://campus-event-planner-backend.vercel.app/](https://campus-event-planner-backend.vercel.app/)

## Backend Infrastructure

The backend is already handled and serves as a plug-and-play API.
- **Backend Repository:** [https://github.com/MadhavDhall/campus-event-planner-backend](https://github.com/MadhavDhall/campus-event-planner-backend)

## Getting Started

1. Clone this repository.
2. Ensure you have Node.js installed.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the frontend development server:
   ```bash
   npm run dev
   ```

Note: Make sure to check the `.env.local` or `.env.example` file to verify that the `VITE_API_URL` points to the deployed backend.
